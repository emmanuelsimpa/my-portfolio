Learn how to AutoMate Chrome Dino

To play this game you have to search chrome://dino to play this game offline and online.

So to automate dino you need to install python3 in your system and then you need to install some package:

  • pyautogui
  • PIL

So firest we need to install pyautogui and PIL into our system by using give command 

pip install pyautogui
pip install pillow

So what steps we are going to take to implement this is here:

  1. First of all, we need to capture the screenshot of the chrome dino game screen on greyscale.(Black and White, because image processing will be good on greyscale).
  2. The we just want to add some kind of rectange on this image for cactus so that when we get those pixels of the rectangle near to dino the accordingly I can write the logic to jump dino
  3. The same goes for birds, will make a rectangle but little lighter than a cactus rectangle. (to differentiate both rectangles) and when we found this rectangle is near to dino then we will press down key to duck the Dino
  4. That's it, after this you will see your chrome dino game is automated using python
